STM Studio
===========

It is a debug tool (don't confuse with STM32 Cube IDE) that can be run in parallel, with STM32 Cube IDE or any IDE.

Each Comp
the STM32 Cube IDE produce .elf project, which contains variables tags. 

You need to import an elf file, and STM Studio will recognize all the tags.
